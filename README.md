# ![App Icon](src/assets/weather-icon.svg) Weather App

## Description

![react](src/assets/react-dark.svg) React app that lets users search for cities and view weather data using the Open-Meteo API. The app features live city search suggestions and dynamic weather updates.

## Features

🏢 Search for cities

⛅ Fetch current and hourly weather data

📱Mobile responsive design

## ⚙️ Technologies

![ React ](src/assets/react-dark.svg) React

• React Query

• Axios

![css](src/assets/css.svg) CSS

![api](src/assets/apiblueprint.svg) Open-Meteo API

##  ![How](src/assets/task-finger-show.svg) How to Use

⌨️ Type a city in the search box.

🪟 🌥️ View current and hourly weather data.

## ![api](src/assets/apiblueprint.svg) API

This app uses the Open-Meteo API for weather data and geocoding.
